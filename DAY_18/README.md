# 🚀 Day 18 – JavaScript API & LocalStorage Projects

## 📅 100 Days of Code Challenge

Today was focused on building real-world JavaScript applications using **DOM Manipulation, Fetch API, Async/Await, Local Storage, and REST APIs**.

## ✅ Projects Completed

### 🔹 GitHub Profile Finder

* Search GitHub users
* Fetch profile data using GitHub API
* Display avatar, followers, following, repositories, and profile link
* Loading & error handling

### 🔹 Users Directory

* Fetch users from JSONPlaceholder API
* Search users by name or email
* Delete users dynamically
* Dynamic rendering with DOM

### 🔹 Expense Tracker

* Add and delete expenses
* Calculate total expenses
* Persistent storage using LocalStorage
* Keyboard shortcuts (Enter key)

### 🔹 Movie Search App

* Search movies using OMDb API
* Display movie poster, title, year, and type
* Handle invalid searches gracefully

### 🔹 Todo App

* Add, delete, and mark todos as completed
* Undo completed tasks
* Track total, completed, and remaining tasks
* Store todos in LocalStorage

### 🔹 Weather App

* Search weather by city
* Fetch live weather using wttr.in API
* Display temperature, humidity, wind speed, and weather description
